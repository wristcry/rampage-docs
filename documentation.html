Plugins


Interception.ClothingWeight

Изменение гравитации/скорости/прыжков при ношении определенного сета брони


Config

  • default_gravity_multiplier - дефолт модификатор гравитации (1 - дефолт в самой игре), будет ставится если у чела нет брони(донатной или вообще)
  • default_speed_multiplier - дефолт модификатор скорости (1 - дефолт в самой игре), будет ставится если у чела нет брони(донатной или вообще)
  • default_jump_multiplier - дефолт модификатор прыжка (1 - дефолт в самой игре), будет ставится если у чела нет брони(донатной или вообще)
  • armor_sets - список сетов
  • backpack_id - айди рюкзака
  • face_id - айди какой-то хуйни (я хз че эт серьезно, но это элемент одежды у игрока)
  • glasses_id - айди очков
  • hat_id - айди шлема
  • mask_id - айди маски(я пенис у меня маска ХАХАХАХАХ)
  • pants_id - айди штанов(за 40 гривен)
  • shirt_id - айди рубахи
  • vest_id - айди броника
  • gravity_multiplier - модификатор гравитации при одетом сете
  • speed_multiplier - модификатор скорости при одетом сете
  • jump_multiplier - модификатор прыжка при одетом сете

Interception.KillFeed

Список убийств/смертей в правом верхнем углу экрана


Config

  • effect_id - айди эффекта(не трогать блять)
  • effect_key - айди ключа эффекта(не трогать блять)
  • buffer_max_update_time_seconds - если в течении указанного тут времени не происходит ни одной смерти - юи исчезает

Interception.Kits

?????? Киты лол (кста юи впадлу делать если честно(но если прям надо то сделаю))


Config

  • display_all_kits - показывать при вводе /kits все киты(при false показывает только те на которые есть пермишн либо есть пермишн kits.creative)
  • remove_cooldowns_on_kit_remove - удалять из бд все кулдауны связанные с китом при его удалении (чтобы не засирать базу данных)
  • kit_list_color - цвет сообщения при вводе /kits
  • kits - список китов
  • name - название кита
  • cooldown - кд в секундах
  • reset_cooldown_on_death - убирать кд при смерти игрока
  • experience - кол-во экспы выдаваемой при получении кита
  • items - список предметов выдаваемых при получении кита
  • id - айди предмета
  • amount - количество предметов(если это магазин или коробка с патронами - то кол-во патронов)
  • durability - прочность предмета(если она у него есть)
  • sight - прицел(если оружие)
  • tactical - рукоятка
  • grip - я забыл что это за обвес
  • barrel - глушитель, etc.
  • magazine - магазин
  • firemode - режим стрельбы
  • fuel - кол-во бензина(если это канистра)

Permissions

  • kits.kadd - /kadd
  • kits.kdel - /kdel
  • kits.kexp - /kexp
  • kits.kiadd - /kiadd
  • kits.kit - /kit
  • kits.kitm - /kitm
  • kits.list - /kits
  • kits.kresetcd - /kresetcd (ненужный нигде пермишн)
  • kits.creative - пермишн позволяющий использовать все киты
  • kit.[kit_name] - пермишн для конкретного кита

Commands

  • /kadd [name] - создать пустой кит
  • /kdel [name] - удалить кит
  • /kexp [name] [amount] - установить кол-во выдаваемой экспы в ките
  • /kiadd [name] [cooldown] - создать кит со списком предметов созданным на основе инвентаря
  • /kit [name] - получить кит
  • /kitm [name] - добавить в кит все предметы из инвентаря
  • /kits - список доступных китов
  • /kresetcd [name] [player] - убрать существующий кулдаун кита для конкретного игрока(эта команда вызывается только с консоли)

Interception.TPA

тпа с юишкой


Config

  • effect_id - айди эффекта(не трогать блять)
  • effect_key - ключ эффекта(не трогать блять)
  • use_plugin_keys - юзать плагин кеи вместо обычных кнопок(ну вот эти вот точка, запятая и т.д.)(ваще не трогай лучше нахуй)
  • accept_key_index - индекс кнопки для принятия(вот поэтому верхнее лучше не трогать)(ЭТО КСТАТИ ТОЖЕ)
  • decline_key_index - индекс кнопки для отмены(тож не трогать)
  • tpa_cooldown_seconds - кд на запрос тп
  • accept_cooldown_seconds - кд на принятие тп
  • active_time_seconds - время которое запрос будет считаться активным

Permissions

  • interception.tpa.tpa - /tpa

Commands

  • /tpa [player] - отправить запрос на телепортацию

StuntFox.PvPShop

ссылочки через команды


Config

  • initial_balance - начальный баланс
  • items - список предметов
  • item_id - айди предмета
  • buy_cost - стоимость покупки предмета
  • sell_cost - стоимость продажи предмета
  • vehicles - список транспорта
  • vehicle_id - айди транспорта
  • cost - цена покупки транспорта

Permissions

  • sf.pvpshop.balance - /balance
  • sf.pvpshop.buy - /buy
  • sf.pvpshop.buyv - /buyv
  • sf.pvpshop.cost - /cost
  • sf.pvpshop.costv - /costv
  • sf.pvpshop.sell - /sell
  • sf.pvpshop.setbalance - /setbalance

Commands

  • /balance - баланс
  • /buy [id] - купить предмет
  • /buyv [id] - купить транспорт
  • /cost [id] - стоимость предмета
  • /costv [id] - стоимость транспорта
  • /sell [id] - продать предмет
  • /setbalance [amount] [player] - установить баланс игроку
